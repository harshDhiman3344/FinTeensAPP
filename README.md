# 💰 FINTEENS - Financial Literacy Platform for Teens

> Interactive, gamified learning platform for teens to master financial concepts through lessons, quizzes, and progress tracking.

**HackIndia Spark 7 - North Region** | Team: Coders Against Humanity

---

## 🎯 Problem Statement

Teenagers lack basic financial literacy, leading to poor money decisions. FINTEENS makes finance education interactive and engaging through gamification.

## 🎓 Project Aim

- Make financial education accessible and engaging for teens
- Build a gamified experience with XP, achievements, and daily quests
- Cover Banking, Budgeting, Investing, and Credit Management
- Track progress in real-time

---

## 📚 How It Works

1. **Select a Course** - Banking Basics, Budgeting 101, Investing Fundamentals, Credit & Debt
2. **Complete Lessons** - Interactive, bite-sized modules
3. **Answer Quizzes** - Multiple-choice questions
4. **Earn XP** - 10 XP per correct answer, resets daily
5. **Track Progress** - Visual progress bars and daily quests

### Key Features
- 🎮 Gamification with points, hearts, and achievements
- 📊 Daily XP system with auto-reset
- 📱 Fully responsive design
- ⚡ No signup required - instant access

---

## 🛠️ Tech Stack

**Frontend**: React 19, Next.js 16.2, TypeScript, Tailwind CSS, Shadcn/ui, Zustand  
**Backend**: Next.js Server Actions, Drizzle ORM, Mock Data System  
**Tools**: Bun, ESLint, Prettier, Turbopack

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ or Bun

### Setup
```bash
git clone https://github.com/HackIndiaXYZ/hackindia-spark-7-north-region-coders-against-humanity.git
cd finteens
bun install
bun run dev
```

Open `http://localhost:3000`

### Build
```bash
bun run build && bun run start
```
